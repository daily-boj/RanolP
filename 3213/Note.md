## 문제 분석

피자를 `3/4`, `1/2`, `1/4` 조각을 먹는데, '한 조각'으로 붙어 있어야 합니다.

## 풀이

- `3/4` 1개가 있다면 `1/4` 1개 하고 결합합니다.
- `1/2` 1개가 있다면 `1/2` 1개 혹은 `1/4` 2개와 결합합니다.
- 이후 결합하지 않은 `1/4` 은 같은 `1/4` 3개와 결합합니다.

의사 코드로 나타내면 다음과 같습니다.

```pseudo-node
if '3/4' > 0 {
  '1/4' -= '3/4';
  pizzas += '3/4';
}
if '1/2' > 0 {
  pizzas += '1/2' / 2;
  if '1/2' % 2 != 0 {
    '1/4' -= 2;
  }
}
if '1/4' > 0 {
  pizzas += ceil('1/4');
}
```
