## 문제 분석

뒤집었을 때 중복되는 문자열을 찾는 문제입니다.

## 풀이

문자열 리스트에 그 리스트의 각 원소를 뒤집은 문자열 리스트를 합칩니다.
리스트 전체를 순회하며 개수가 2개인 문자열을 찾으면 됩니다.
가운데 글자를 찾는 것이고, 길이가 홀수이므로 어떤 문자열을 찾아도 상관 없습니다.
